# Partner logos (optional)

Logos on the Sponsorship page are loaded from:

1. **Google Favicon** – used by default so every brand shows a logo image.
2. **Clearbit** – used as fallback when available for higher-resolution logos.

To use **your own logo files** (e.g. official brand assets), add PNG or SVG files here with this naming:

- `icicibank-com.png`
- `sbicard-com.png`
- `scotiabank-com.png`
- `cibc-com.png`
- `rbc-com.png`
- `enterprise-com.png`
- `avis-com.png`
- `hertz-com.png`
- `century21-com.png`
- `royallepage-ca.png`
- `remax-ca.png`
- `bell-ca.png`
- `rogers-com.png`
- `timhortons-ca.png`
- `starbucks-com.png`

Use the domain with dots replaced by hyphens. The app will use these files when the logo component is updated to check this folder first.
